# Gourmet Cafeteria

## A Cafeteria Management Application built on Rails.

## FEATURES:

### 1. Support multiple user roles: 
##### a) Customer
##### b) Billing-Clerk
##### c) Owner

### 2. Customers can:
##### - see the menu
##### - sign up/ sign in
##### - add items to their cart
##### - place order
##### - see their orders history
##### - see the status of their orders
